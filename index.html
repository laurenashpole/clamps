Clamps Demo

Clamps is a lightweight JavaScript module that mimics the -webkit-line-clamp CSS property on browsers that don't natively support it (Can I use details). To get the most out of this demo, open the page on a browser without support like Firefox or IE and compare the results with Chrome or Safari.
